# Social Media Web Application

A full-stack social media web application built using the MERN stack that enables users to connect, share content, and communicate in real time with a secure and scalable system.

## Features
- User authentication and authorization
- Create, edit, and delete posts (text & images)
- Follow and unfollow users
- Friend requests and user discovery
- Real-time chat with notifications
- Optimized image storage and responsive UI
